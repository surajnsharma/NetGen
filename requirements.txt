# macOS Qt Platform Plugin Fix:
# The run_tgen_client.py automatically sets QT_PLUGIN_PATH for macOS to resolve
# the "Could not find the Qt platform plugin 'cocoa'" error.
#
# If you encounter Qt platform plugin issues:
# 1. Ensure PyQt5 is installed from Qt's official wheels (not source)
# 2. Check that QT_PLUGIN_PATH points to: <venv>/lib/python3.x/site-packages/PyQt5/Qt5/plugins
# 3. Alternative: Use the ostg_client_wrapper.sh script which sets the environment variable
#
# Installation command for PyQt5 with proper macOS support:
# pip install --find-links https://download.qt.io/snapshots/ci/pyqt5/5.15/wheels/ PyQt5
#
# Alternative GUI frameworks (if PyQt5 issues persist):
# - PyQt6: pip install PyQt6 (requires code changes from PyQt5 to PyQt6)
# - PySide6: pip install PySide6 (requires code changes from PyQt5 to PySide6)
